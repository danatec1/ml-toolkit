## 5. Usage Examples

```bash
$ python cli.py list
등록된 모델 (7개):
  - pallets: 물적자원 - 총파렛트수 예측
  - workers: 인적자원 - 중근조 인원수 예측
  ...
```

### 전체 예측
```bash
$ python cli.py predict-all --input 200000

  workers  | 중근조인원수       | 64.71
  ...
  총 인원   |                 | 406.30
```
